# Built-In Image Generation

Use this reference when paragraph images are part of the WeChat draft.

## Default Rule

Use the built-in image generation path for real bitmap visuals. Do not ship placeholder cards made only from rectangles, circles, gradients, or generic diagrams.

Acceptable paragraph image:

- The base visual is generated by the built-in image model or official configured image provider.
- The scene matches the paragraph: field site, hardware, workflow, risk boundary, or decision frame.
- Alibaba orange `#ff6a00` is used as an accent, not as a full-page wash.
- Chinese text is readable on phone and desktop.

Unacceptable paragraph image:

- Pure ImageMagick/SVG/HTML geometry pretending to be an illustration.
- Generic server room, random drone, abstract glowing AI brain, or decorative shapes unrelated to the paragraph.
- Text too small, garbled, low contrast, crowded, or close to edges.
- Same scene reused with only the title changed.

## Text Fidelity

Image models may distort Chinese text. Prefer this two-step path for production drafts:

1. Generate a text-free or low-text scene with the built-in image model.
2. Add exact Chinese title/callout locally with deterministic typography.

The local overlay is allowed only when the generated base scene is meaningful. It is not a license to replace AI images with simple shape cards.

Recommended overlay:

- Body image canvas: `1200 x 900`
- Cover: `2100 x 900`
- Safe margin: at least `72px` for body images and `120px` for cover
- Accent: Alibaba orange `#ff6a00`
- Text: one title of 6-12 Chinese characters and one callout of 12-20 Chinese characters
- Keep text away from the bottom-left cover safe zone because WeChat may overlay the article title

## Prompt Requirements

Write each prompt before generation. Include:

- Asset role: cover, field hook, mechanism, workflow, boundary, decision
- Scene subject: DJI M300 RTK, Jetson TX2/Manifold 2-G, edge inference, disaster response, inspection corridor, command tablet, etc.
- Visual language: industrial editorial photography + blueprint overlay, realistic hardware, grounded lighting
- Palette: off-white/charcoal/steel gray with Alibaba orange accent
- Text policy: no embedded text if local overlay will be applied
- Negative guidance: no cartoons, no generic robot, no abstract glowing brain, no illegible dashboard text

## WeChat Readback QA

After saving a draft, use the temporary preview URL or equivalent readback and verify:

- Title exists.
- Author follows the account default; do not fill author unless requested.
- `没时间看全文版` exists and has the same count as paragraph images.
- Body paragraph count matches the outline.
- Body image count matches the outline.
- Body Chinese character count is within the target range.
- Each required paragraph sentence is present.
- Each generated image appears in order.

If any item fails, report `FAILED` and fix before claiming completion.
